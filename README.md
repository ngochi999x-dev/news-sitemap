# Deployment Instructions

1. **Clone the Repository**:  Use the following command to clone the repository to your local machine:
   ```bash
   git clone https://github.com/ngochi999x-dev/news-sitemap.git
   ```

2. **Navigate to the Project Directory**:  Change into the project directory:
   ```bash
   cd news-sitemap
   ```

3. **Install Dependencies**: Before running the application, ensure all dependencies are installed:
   ```bash
   npm install
   ```

4. **Build the Project**: Compile the project using:
   ```bash
   npm run build
   ```

5. **Run the Application**: After building the project, you can run it with:
   ```bash
   npm start
   ```

6. **Access the Application**: Open your web browser and navigate to `http://localhost:3000` to view the application.

## Project Documentation

This project is designed to generate XML sitemaps for news content automatically. It follows best practices for sitemap generation and aims to help websites enhance their SEO.

### Features
- Automatically generates news sitemaps
- Supports various news content formats
- Easy to configure and deploy

### Technologies Used
- Node.js
- Express
- XML generation libraries
